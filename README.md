## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This application is a stopwatch application. Shows developers how to use the Timer class. A Label is used so that the user can see the second. Below it is three TintedButtons to start, stop and restart the stopwatch. To use the Timer class, we first created an object from this class. Then we used the scheduledTimer method to initialize the counter. This method first requests a time slot. We set it to 1 because we want it to count down at 1-second intervals. Then you need a target. This is asking us to specify the ViewController we are in. That's why we used self-statement. Another thing he wants is the selector. We have given this structure an objective c function. This allowed the counter to count backward. He wonders if any information will come from the user with the UserInfo feature. We have given the expression nil to indicate that this will not happen. Finally, it asks whether to repeat the operation with repeats. We start the counter by calling it true. The pause button pauses and resumes the counter. Disabled using the Pause button when the counter is at 25 and 0. In this way, the counter will only start with the Start button and will not continue to work when the counter is at 0. Finally, we reset the counter and logic expressions with the Restart button. In this way, the application starts again without any problems.
